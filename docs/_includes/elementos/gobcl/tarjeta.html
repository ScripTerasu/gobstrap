Tarjeta Full

A partir de un elemento .tarjeta se pueden construir otro elementos modulares más complejos. Por ejemplo, agregándole .tarjeta-full, se crea un elemento modular que sirve para presentar información con una fotografía destacada, como una noticia o un anuncio importante. La variación de la fotografía puede ser variable, pero se recomienda usar imágenes apaisadas para no hacer crecer mucho el elemento.

Tarjeta horizontal - 12 columnas

Así mismo se puede usar la clase .tarjeta-horizontal junto a .tarjeta para un módulo con fotografía destacada y título, pero sin texto. En este caso, la fotografía debe mantener una relación de aspecto de 16:9 (1920x1080, 1280x720, etc). Si se quiere usar otro formato, se puede modificar el alto del módulo, pero se recomienda usar el predeterminado para evitar problemas en distintos dispositivos.

Atención con el uso

Debido a sus características de alto y ancho, los módulos .tarjeta-full y .tarjeta-horizontal están diseñadas para ser usadas en condiciones específicas: .tarjeta-full dentro de un .col-md-12 y .tarjeta-horizontal dentro de .col-md-12 o .col-md-8. Para usarlo en otras opciones de grilla, es necesario revisar y adaptar -cuando corresponda- los estilos en gobstrap.less.
