Lista colapsable

Puedes hacer una lista colapsable que empieza oculta y se muestra al apretar el elemento que la activa y hace de encabezado y título de la lista. La lista se muestra con un alto máximo de 250px y pasado ese largo se muestra un scroll. Este elemento funciona con JavaScript.
